# Shiv_Enterprises Personal Site (MERN scaffold)

This workspace contains a minimal MERN scaffold: a small Express backend and a Vite + React frontend.

Quick start (two terminals):

```powershell
cd Backend
npm install
npm start

# in a second terminal
cd frontend
npm install
npm run dev
```

Backend runs on `http://localhost:4000` and exposes `POST /api/contact`.
